# Correction Learning Template

Use this when a user provides the final diagnosis after an incorrect or incomplete interpretation.

```text
Correction note:
- Initial impression:
- Final diagnosis:
- Evidence for final diagnosis: pathology / surgery / follow-up / expert report / user feedback / literature
- Missed or underweighted imaging clue:
- Mimic/pitfall:
- Rule for next time:
- Suggested literature/reference to add:
```

Privacy rule: store only de-identified lessons. Do not store names, accession numbers, exact dates, or patient-identifying images.
